# Fujiwara API

## Installation

### For MacOS, Linux, Win

```shell
    docker-compose build
```

## Development

```shell
    docker-compose up
```

## Docs API

Swagger UI at `http:localhost:3000/swagger`

[How to docs api](https://github.com/Surnet/swagger-jsdoc/blob/master/docs/GETTING-STARTED.md)
