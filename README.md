# Fujiwara API

## Installation

### For MacOS

```shell
    npm install
```

### For Linux, Win

```shell
    npm install --no-optional
```

## Development

```shell
    npm run dev
```

## Docs API

Swagger UI at `host/swagger`

[How to docs api](https://github.com/Surnet/swagger-jsdoc/blob/master/docs/GETTING-STARTED.md)

## Style Code

```shell
    npm run format
```
